# AgRup Media Website

Aktuelle Next.js-Version der AgRup-Media-Website.

## Lokal starten

```bash
npm install
npm run dev
```

Danach im Browser öffnen:

```text
http://localhost:3000
```

## Vor dem Hochladen prüfen

```bash
npm run lint
npm run build
```

## Wichtig für GitHub

Diese Ordner nicht hochladen:

- `node_modules`
- `.next`
- `out`

Sie werden durch `.gitignore` ausgeschlossen und bei Bedarf lokal neu erzeugt.
